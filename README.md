### Running Locally

Clone the project and make it your working directory:
```sh
git clone https://github.com/lowzijian/Naluri-technical-assessment-FE.git
cd Naluri-technical-assessment-FE
```

Install Dependecies:
```sh
npm install
```

Run the project:
```sh
npm run start
```
